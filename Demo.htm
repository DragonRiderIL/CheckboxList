The jQuery Checkbox List Plugin

The jQuery Checkbox List Plugin was designed to accept a collection of groups of lists to be presented to the user as checkboxes. Individual items may be disabled or pre-selected, or entire groups my be disabled on render (showing options that are selected or not, but not available for change). The list of groups takes the size of the host container and wraps left to right according to size. All groups of lists are sized the same so that they column nicely while wrapping (note: groups width will be the width of the size of the largest list item). Plugin will return an object consisting of groups with all selected items, only selected items are returned.

Source Code Example

This plugin can be called on any container as follows:

$('#OptionListDiv').CheckboxList({ SourceUrl: 'OptionListServer.asmx/GetList' });
$('#OptionListDiv').CheckboxList({ SourceData: data });

Prerequisites

This plugin has been tested using the following configuration.

  • jQuery 1.6
  • jQuery UI 1.8
  • jquery.tmpl.min.js
  • json2.js

API Methods

$(.selector).CheckboxList("GetSelectedItems")
This method will return a collection of groups with list items that have been selected, it returns ONLY groups that have items selected and those selected items.

.data-group
Customizes the group label and group container.

.data-item
Customizes the look of individual data items within the group.

.data-item-disabled
Customize the look of disbled data items.

.data-error
Customize the look of the error message shown if the web service cannot be accessed.

Expected Data Format

Following is the expected data format(whether by object or webservice) and field definitions: 

                        [Object with one or more group objects]
                            GroupName = {Title for Group - will be displayed, can be null if no title desired.},
                            GroupId = {Id for the group},
                            DisableGroup = {bool - If true, will render all items disabled},
                            [Items] - Collection of one or more list items
                            {
                                OptionName = {Title for the Checkbox Items - Will be displayed.},
                                OptionId = {Id for the Checkbox item - must be unique by group},
                                Selected = {bool - if true, item will be pre-selected and shown checked on display},
                                Enabled = {bool - if false, will render the Checkbox item as non-selectible}
                            }
                        [Object End]
